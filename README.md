## Project Overview

This project is a Shared Expense Management Application developed for the Spreetail Software Developer Assignment.

The application imports expense data from CSV files, detects anomalies, generates reports, and calculates balance summaries.

## Features

* CSV Import
* Expense Analysis
* Anomaly Detection
* Import Report Generation
* Balance Summary Generation
* SQLite Database
* Prisma ORM
* Express Backend

## Reports

* IMPORT_REPORT.md
* BALANCE_REPORT.md
* SCOPE.md
* DECISIONS.md
* AI_USAGE.md

## Tech Stack

* Node.js
* Express.js
* Prisma ORM
* SQLite
